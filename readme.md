# Valkyrie

Valkyrie is an interpreted language built using C++. The language aims to provide a simple and intuitive syntax for developers.

## Features

- Simple and intuitive syntax
- Interpreted language
- Built using modern C++

## Building

To build the Valkyrie interpreter, you will need a C++ compiler that supports C++17.

g++ -std=c++17 -o valkyrie main.cpp -->
